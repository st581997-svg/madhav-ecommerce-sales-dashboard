# 📊 Madhav E-Commerce Sales Dashboard (Power BI)

## 📌 Project Overview

This project presents an interactive **E-Commerce Sales Dashboard** built using **Power BI** to analyze sales performance, profit trends, customer payment behavior, and product category insights.

The dashboard provides actionable business insights through KPI tracking, monthly trend analysis, and category-wise performance evaluation.

---

## 🎯 Business Objective

To analyze sales data and identify:

- Revenue performance
- Monthly profit trends
- Most used payment methods
- Top-performing product categories
- Sub-category sales contribution

---

## 📈 Key Performance Indicators (KPIs)

| Metric | Value |
|--------|--------|
| 💰 Total Sales | 1.54M |
| 📦 Total Quantity | 4202 |
| 💵 Total Amount | 302K |
| 📊 Average Sales | 83K |
| 🟢 Total Profit | 21K |

---

## 🔍 Key Insights

- COD is the most preferred payment mode (58%+ contribution)
- November recorded the highest profit
- Printers category generated the highest sales
- Some months showed negative profit trends (business risk indicators)
- Category-wise quantity distribution highlights major revenue drivers
